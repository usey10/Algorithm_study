# Problem2

## 문제 설명

민수는 택배 운송 기사로, 하루에 한번씩 차량을 가득 채워서 운반을 한다.

리스트 `weights`에는 보유한 택배 상품의 무게가 각각 기록되어 있다.

`delivery`는 택배 상품을 모두 배송해야 하는 납기까지 남은 일 수이다.

`delivery`이내에 모든 택배 상품을 배송하려면 필요한 차량의 최소 적재량을 구하시오.

단, 택배 상품은 순서대로 배송해야 한다.

## 매개변수 형식

`weights = [3, 2, 2, 4, 1, 4]`

`delivery = 3`

## 반환값 형식

`6`
