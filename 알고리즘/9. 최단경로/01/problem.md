# Problem1

## 문제 설명

Dijkstra 알고리즘을 구현하세요!

전체 노드의 개수 `N`개가 있고, 각 노드간의 연결이 `edge[i] = [출발노드, 도착노드, 가중치]`로 주어져 있다고 하자.

이 때 `start` 노드에서부터 모든 노드에 도달하기 위한 최단거리를 구하시오.


## 매개변수 예시

`N = 5`

`edge = [[1, 2, 2], [1, 3, 3], [2, 3, 4], [2, 4, 5], [3, 4, 6], [4, 5, 1]]`

`start = 1`

## 출력 예시

`[0, 2, 3, 7, 8]`
