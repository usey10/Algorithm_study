# Problem1

## 문제 설명

0-1 Knapsack 문제를 해결하세요!

당신은 `capacity`만큼의 무게를 담을 수 있는 배낭에 물건을 담고자 한다.

각 물건의 무게는 `weight[i]`, 각 물건의 가치는 `value[i]`로 주어져 있을 때,

배낭에 담을 수 있는 가능한 물건들의 가치의 합 중 최대의 값을 구하시오.


## 매개변수 예시

`capacity = 20`

`weight = [4, 5, 2, 3, 6, 8, 5, 5]`

`value = [5, 2, 6, 7, 1, 3, 4, 6]`

## 출력 예시

`28`
